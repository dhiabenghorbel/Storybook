# Welcome to GP4You Components

As part of HR Access's commitment to new digital trends in the field of Human Resources, GP4You is a web solution with self-service and collaborative process functionalities for employees and managers.

GP4You is built with the latest front-end technologies and has a responsive design that adapts to PCs, smartphones, and tablets, providing more flexibility and improving the employees' experience in interacting with Human Resources processes in companies.
